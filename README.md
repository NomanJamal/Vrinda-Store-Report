# Vrinda-Store-Report
An Excel-based data analysis project on Vrinda Store sales, featuring data cleaning, pivot tables, charts, and an interactive dashboard to generate business insights.
An Excel-based data analysis project on Vrinda Store sales using a dataset from Kaggle. Includes data cleaning, pivot tables, and interactive dashboards.

 **Project Features**
Real-World Dataset
Uses sales data sourced from Kaggle for practical analysis
Data Cleaning & Preprocessing
Handles missing values, removes duplicates, and formats data for accuracy
Pivot Tables & Data Analysis
Efficient data summarization to identify trends and patterns
Interactive Dashboard
Dynamic Excel dashboard with slicers for easy filtering
Sales Trend Analysis
Monthly and yearly performance tracking
Product Performance Insights
Highlights top-selling and low-performing products
Customer Behavior Analysis
Understands buying patterns and preferences
Data Visualization
Clear charts (bar, line, pie) for better understanding
Business Insights Generation
Helps in decision-making using data-driven insights
User-Friendly Interface
Simple and clean layout for easy navigation
